Technical Approach

🔧 New JavaScript-Based Solution:
• JavaScript adds CSS classes: next-only, prev-only, both-visible
• CSS uses these classes to set appropriate justify-content values
• More reliable than CSS-only selectors trying to detect hidden elements

Why the Original Fix Failed

❌ CSS Selector Issues:
• :only-child didn't work because both buttons always exist in DOM
• :has() selector has limited browser support
• Selecting inline styles is unreliable and not well-supported
✅ JavaScript Solution Benefits:
• Reliable across all browsers
• Clear semantic meaning with class names
• Easy to debug and maintain
• Handles all edge cases properly
