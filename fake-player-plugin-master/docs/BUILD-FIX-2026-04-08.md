# Build Fix — April 8, 2026

## Summary

Fixed compilation errors in the FakePlayerPlugin build after the move waypoint system refactor. The build now compiles successfully with no errors.

---

## Issues Fixed

### 1. **Missing `Config.heartbeatEnabled()` Method**

**Error:**
```
[ERROR] /C:/Users/Name/IdeaProjects/fake-player-plugin/src/main/java/me/bill/fakePlayerPlugin/util/StatsHeartbeat.java:[69,20] cannot find symbol
  symbol:   method heartbeatEnabled()
  location: class me.bill.fakePlayerPlugin.config.Config
```

**Root Cause:**
`StatsHeartbeat.java` was calling `Config.heartbeatEnabled()` which didn't exist in `Config.java`.

**Fix:**
1. Added `heartbeatEnabled()` method to `Config.java`:
   ```java
   /** Whether stats heartbeat to Vercel API is enabled. Maps to {@code heartbeat.enabled}. */
   public static boolean heartbeatEnabled() {
       return cfg.getBoolean("heartbeat.enabled", false);
   }
   ```

2. Added corresponding config section to `config.yml`:
   ```yaml
   # Anonymous server stats heartbeat to the FPP Vercel API homepage.
   # POSTs server_id + bot/player counts every 5 minutes for live website stats.
   # No personal data collected. Purely opt-in (disabled by default).
   heartbeat:
     enabled: false
   ```

### 2. **Missing Mojang DataFixerUpper Dependency**

**Error:**
```
[ERROR] C:\Users\Name\.m2\repository\io\papermc\paper\paper-server\1.21.11-R0.1-SNAPSHOT\paper-server-1.21.11-R0.1-SNAPSHOT-mojang-mapped.jar(/net/minecraft/server/level/ServerLevel.class): error: Cannot attach type annotations @org.jspecify.annotations.Nullable to net.minecraft.server.level.ServerLevel.findClosestBiome3d:
  class file for com.mojang.datafixers.util.Pair not found
```

**Root Cause:**
The `paper-server` dependency references classes from Mojang's DataFixerUpper library (used for type annotations in NMS code), but this transitive dependency wasn't being resolved properly.

**Fix:**
Added explicit dependency in `pom.xml`:
```xml
<!-- Mojang DataFixerUpper — required by paper-server for type annotations -->
<dependency>
    <groupId>com.mojang</groupId>
    <artifactId>datafixerupper</artifactId>
    <version>8.0.16</version>
    <scope>provided</scope>
</dependency>
```

## Notes

- The `heartbeat` feature is **disabled by default** (`enabled: false`) — purely opt-in for server owners who want to contribute anonymous stats to the FPP website homepage
- No personal data is collected — only `server_id`, `player_count`, `bot_count`, and `version`
- POSTs to `https://fake-player-plugin.vercel.app/api/heartbeat` every 5 minutes
- Runs asynchronously, never blocks the main thread
- Network errors are silently ignored unless `logging.debug.network` is enabled
